# Redirecting more output
In this level, /challenge/run will once more give you a flag, but only if you redirect its output to the file myflag. Your flag will, of course, end up in the myflag file!
## Flag: pwn.college{8KmOgiLDhrYaxN5EMc8xSgm0Euf.QX1YTN0wyM1UjM2EzW}
commands used:
/challenge/run > myflag
cat myflag
I had the redirect the stout to the file named myflag and then read the myflag file using the cat command to get the output.
