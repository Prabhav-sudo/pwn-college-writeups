# Translating characters
In this level, /challenge/run will print the flag but will swap the casing of all characters (e.g., A will become a and vice-versa).

command used: /challenge/run | tr 'A-Za-z' 'a-zA-z'
In this challenge I had to use the tr command to swap the casing of all the characters.
